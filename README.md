# CRM Client Form Project

Plataforma integral para la gestión de clientes y formularios dinámicos, desarrollada con un enfoque en arquitectura modular y escalabilidad.

## 🚀 Tecnologías utilizadas
* **Frontend:** React + Vite + Tailwind CSS
* **Backend:** Node.js + Express
* **Base de Datos:** MySQL
* **Despliegue:** Netlify (Functions & Hosting)

## 📂 Estructura del Proyecto
* `/client`: Aplicación frontend en React.
* `/server` & `/api`: Lógica de backend y endpoints.
* `/shared`: Tipos y utilidades compartidas.
* `/documentos`: Documentación técnica y registros de cambios.

## 🛠️ Instalación y Configuración
1. Clonar el repositorio.
2. Instalar dependencias: `pnpm install`
3. Configurar variables de entorno en un archivo `.env` (usar `.env.example` como base).
4. Ejecutar en modo desarrollo: `npm run dev`
