KEYBOARD SHORTCUTS & TIPS

DevTools Inspection:
  F12              → Abrir DevTools
  Ctrl+Shift+M     → Toggle responsive view
  Ctrl+Shift+I     → Inspect element (para ver media queries)

Test Touch Targets:
  // Copiar en consola para verificar botones < 44px:
  document.querySelectorAll('button').forEach(btn => {
    const rect = btn.getBoundingClientRect();
    if (rect.height < 44 || rect.width < 44) {
      console.warn('Button too small:', rect.width, 'x', rect.height, btn);
    }
  });

Verify Dark Mode:
  En CSS inspector: busca "dark:" para ver estilos
