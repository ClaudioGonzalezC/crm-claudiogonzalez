5. PADDING CONSISTENTE (1.2rem base)
================================================================================

REGLA: Los headers usan padding base de ~1.2rem en móvil

  px-4         (horizontal: 1rem = 16px en móvil)
  sm:px-6      (horizontal: 1.5rem = 24px en tablet)
  lg:px-8      (horizontal: 2rem = 32px en desktop)

  py-4 sm:py-6 lg:py-8  (vertical escalado igual)

VISUALIZACIÓN:
┌────────────────────────────────────────────┐
│╌╌╌4px╌╌╌ Contenido          ╌╌╌4px╌╌╌│
│╌╌╌4px╌╌╌ Header             ╌╌╌4px╌╌╌│
│╌╌╌4px╌╌╌                     ╌╌╌4px╌╌╌│
└────────────────────────────────────────────┘

MÓVIL (375px):
  px-4 = 16px izq + 16px der = 32px reservados
  Espacio disponible: 375px - 32px = 343px

DESKTOP (1920px):
  lg:px-8 = 32px izq + 32px der = 64px reservados
  Espacio disponible: 1920px - 64px = 1856px

Móvil:   Rediseño de Plataforma Digi...

whitespace-nowrap:
  Impide que "Salir" o "Dashboard" se corten en "Sali..." o "Dash..."
  
  ✓ Correcto: "Salir" (no hay donde cortar palabra corta)
  ✗ Incorrecto: "Sali..." (word-wrap aplicado sin necesidad)

7. RESPONSIVE TEXT HIDING
================================================================================

OCULTAMIENTO PROGRESIVO:

  hidden            ← SIEMPRE oculto
  sm:inline         ← Mostrar en 640px+ (tablets+)
  md:block          ← Mostrar en 768px+ (tablets+)
  lg:inline         ← Mostrar en 1024px+ (desktop+)

EJEMPLO: Botones en Header

MÓVIL (< 640px):
  ┌──────────────────────────────┐
  │ 🏠 │ ⚙️ │ 🚪                │
  │ (solo iconos - ahorra 40px)  │
  └──────────────────────────────┘

TABLET (640px - 1024px):
  ┌──────────────────────────────┐
  │ 🏠 Dashboard │ ⚙️ │ 🚪 Salir  │
  │ (iconos + texto pequeño)     │
  └──────────────────────────────┘

DESKTOP (1024px+):
  ┌──────────────────────────────────────┐
  │ 🏠 Dashboard │ ⚙️ Ajustes │ 🚪 Salir  │
  │ (todo visible, espaciado normal)    │
  └──────────────────────────────────────┘

CSS:
  <span className="hidden sm:inline">Dashboard</span>
  └─ Oculto por defecto, visible desde sm (640px)
